# Excel Sales Dashboard Project Progress

## Project Overview

This project was completed as part of my Data Analytics learning roadmap. The objective was to perform an end-to-end sales analysis using Microsoft Excel, starting from raw data cleaning to building an interactive dashboard and documenting the project on GitHub.

---

## Day 1 – Project Setup

- Created project folder structure
- Downloaded the Tableau Sample Superstore dataset
- Preserved the original dataset (`Raw_Data.xlsx`)
- Created a working copy (`Working_File.xlsx`)
- Initialized Git repository
- Created GitHub repository

---

## Day 2 – Data Exploration

- Explored the dataset structure
- Identified 21 columns and 9,994 records
- Understood the business meaning of each column
- Created documentation and project notes
- Converted the dataset into an Excel Table

---

## Day 3 – Data Cleaning

Completed data quality assessment:

- Checked for duplicate records
- Identified missing Postal Code values
- Verified no negative shipping days
- Verified no invalid discount values
- Verified no negative sales values
- Verified valid quantity values

Created new analysis columns:

- Shipping Days
- Year
- Month Name
- Quarter
- Profit Margin

Applied appropriate formatting to dates, currency, and percentage fields.

---

## Day 4 – Exploratory Data Analysis (EDA)

Calculated key business metrics:

- Total Sales

- Average Order Value

- Total Quantity Sold

Created Pivot Tables to analyze:

- Sales by Region
- Profit by Category
- Monthly Sales Trend
- Sales by Segment
- Top Customers
- Top Products
- Profit by Sub-Category
- Sales by State
- Avg Discount by Category
- Avg Profit Margin by Category


Documented business insights from each analysis.

## Key Business Insights

- West region generated the highest sales.
- Technology was the most profitable category.
- November recorded the highest monthly sales.
- Certain product sub-categories generated negative profits despite strong sales.
